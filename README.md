# BrieLLa.YB Portfolio

Personal portfolio website showcasing projects, achievements, and professional information.

## 🎨 Features

- **Modern Landing Page** with loading animation (session-based)
- **Interactive Lanyard ID Card** - AI-generated 3D component with:
  - Glossy logo pin with metallic frame effect
  - Realistic rope/lanyard texture with gradient
  - Transparent ID card with glassmorphism effect
  - 3D depth and shadow effects
  - Static display (non-interactive)
- **Project Showcase** with gradient cards and live demo links
- **Achievement Gallery** with certificate lightbox
- **Contact Form** with Formspree integration
- **Responsive Design** - Mobile-first approach
- **Smooth Animations** - AOS library integration
- **Active Navigation States** with animated underlines

## 🛠️ Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla)
- AOS (Animate On Scroll)
- Bootstrap Icons

## 🎯 Highlights

### Lanyard Component
The profile photo in the biodata page features a unique AI-designed lanyard ID card component:
- **Pin Design**: Logo embedded in metallic circular frame with 3D inset shadows
- **Glossy Effect**: Gradient overlay creating realistic shine
- **Rope/Lanyard**: Purple gradient with fabric-like texture and shadow depth
- **ID Card**: Glassmorphism with backdrop blur and subtle border glow
- **Composition**: Pin positioned above rope (z-index layering) for realistic pinned effect
